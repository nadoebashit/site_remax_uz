# RE/MAX Kazakhstan Landing Page

## Обзор
Современный, адаптивный лендинг для RE/MAX Kazakhstan, ориентированный на привлечение франчайзи. Дизайн выполнен в фирменном стиле RE/MAX с использованием брендовых цветов (красный, синий, белый) и логотипов.

## Инструкция по развертыванию

### Вариант 1: Простое развертывание на хостинге
1. Загрузите все файлы и папки из этого архива на ваш хостинг через FTP или панель управления хостингом
2. Убедитесь, что файл `index.html` находится в корневой директории вашего сайта
3. Сайт должен быть доступен по вашему домену

### Вариант 2: Развертывание с использованием GitHub Pages
1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы из этого архива в репозиторий
3. В настройках репозитория включите GitHub Pages, выбрав ветку main и корневую директорию
4. Сайт будет доступен по адресу: https://[ваш-username].github.io/[имя-репозитория]

### Вариант 3: Развертывание на Netlify
1. Зарегистрируйтесь на [Netlify](https://www.netlify.com/)
2. Перетащите папку с файлами лендинга в область загрузки на Netlify
3. Сайт будет автоматически развернут и вы получите URL для доступа

## Настройка и изменение контента

### Изменение текста
1. Откройте файл `index.html` в любом текстовом редакторе
2. Найдите соответствующий раздел по комментариям (например, `<!-- Hero Header Section -->`)
3. Измените текст внутри HTML-тегов

### Изменение изображений
1. Подготовьте новые изображения (рекомендуется оптимизировать их для веба)
2. Замените файлы в папке `images/` сохраняя те же имена файлов
3. Или измените пути к изображениям в файле `index.html`

### Изменение стилей
1. Откройте файл `css/styles.css`
2. Для изменения цветовой схемы отредактируйте переменные в начале файла (`:root`)
3. Для изменения конкретных элементов найдите соответствующий селектор
