# RE/MAX Kazakhstan Landing Page - Документация

## Обзор проекта

Современный, адаптивный лендинг для RE/MAX Kazakhstan, ориентированный на привлечение франчайзи. Дизайн выполнен в фирменном стиле RE/MAX с использованием брендовых цветов (красный, синий, белый) и логотипов.

## Структура проекта

```
remax-landing/
├── css/
│   └── styles.css       # Основные стили с адаптивным дизайном
├── js/
│   └── main.js          # JavaScript функциональность и анимации
├── images/              # Оптимизированные изображения RE/MAX
├── index.html           # Основная HTML структура
└── documentation.md     # Документация проекта
```

## Технические особенности

### 1. Дизайн и визуальные элементы

- **Цветовая схема**: Использованы официальные цвета бренда RE/MAX:
  - Красный: `#e1251b`
  - Синий: `#00285e`
  - Светло-синий: `#0066cc`
  - Белый: `#ffffff`

- **Типографика**: Использован шрифт Montserrat для чистого, современного вида

- **Адаптивность**: Полностью адаптивный дизайн с тремя контрольными точками:
  - Десктоп: 1024px и выше
  - Планшет: 768px - 1023px
  - Мобильный: до 767px

### 2. Разделы страницы

1. **Hero Header**
   - Логотип RE/MAX слева
   - Контактный телефон справа
   - Заголовок с призывом к действию
   - Форма заявки (имя, телефон)
   - Анимированное изображение воздушного шара RE/MAX

2. **О компании RE/MAX**
   - Заголовок и описание
   - Четыре блока статистики с иконками
   - Анимация появления при прокрутке

3. **Преимущества франшизы**
   - Шесть блоков с иконками и описанием
   - Анимация при наведении

4. **Процесс открытия франшизы**
   - Четыре шага с нумерацией и иконками
   - Визуальные соединители между шагами

5. **Технологии RE/MAX**
   - Три блока с описанием технологий
   - Иконки и стилизованные контейнеры

6. **Инвестиционные ожидания**
   - Два блока с информацией об окупаемости и стартовых инвестициях

7. **Финальный призыв к действию (CTA)**
   - Форма заявки
   - Контактный телефон
   - Ссылки на мессенджеры (WhatsApp, Telegram)

8. **Футер**
   - Логотип RE/MAX
   - Копирайт

### 3. Интерактивные элементы

- **Анимации при прокрутке**: Элементы плавно появляются при прокрутке страницы
- **Hover-эффекты**: Кнопки, блоки и иконки реагируют на наведение курсора
- **Формы**: Стилизованные поля ввода с анимацией фокуса
- **Плавная прокрутка**: При клике на навигационные элементы

### 4. Оптимизация

- **Изображения**: Все изображения оптимизированы для быстрой загрузки
- **CSS**: Структурированные стили с использованием переменных CSS
- **JavaScript**: Минимальный и эффективный код для интерактивности

## Инструкции по обновлению

### Изменение контента

1. Откройте файл `index.html`
2. Найдите соответствующий раздел по комментариям (например, `<!-- Hero Header Section -->`)
3. Внесите необходимые изменения в текст или структуру

### Изменение стилей

1. Откройте файл `css/styles.css`
2. Для изменения цветовой схемы отредактируйте переменные в начале файла (`:root`)
3. Для изменения конкретных элементов найдите соответствующий селектор

### Добавление новых изображений

1. Оптимизируйте новые изображения для веб
2. Поместите их в папку `images/`
3. Обновите ссылки в HTML-файле
